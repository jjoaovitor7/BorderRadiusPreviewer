Border Radius Previewer

Olá! Esse é o meu Border Radius Previewer.


Como funciona?

É simples, digitando os valores nos campos setados inicialmente em 0.


Obs.1: Para saber qual o canto que você quer alterar é necessário deixar o campo em branco, ou seja, apagando todos os caracteres dele.


Obs.2: Ao clicar em resetar, o preview é resetado e é apresentado um código com os 3 tipos de implementação.

border-radius: 0px 0px 0px 0px;
